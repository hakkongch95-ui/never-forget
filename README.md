# NEVER FORGET

절대 잊으면 안 되는 일을 강제로 기억시키는 Windows 데스크탑 앱.

무슨 수를 써서라도 당신이 잊지 못하게 만든다.

---

## 기능

- 마감 카운트다운 (실시간)
- 에스컬레이션 알림 (토스트 → 팝업 → 풀스크린 차단)
- 기억 검증 (직접 타이핑으로 확인)
- 위협 메시지 (무시할수록 강도 상승)
- 완수 확인

## 에스컬레이션 단계

| 시점 | 방식 |
|---|---|
| 마감 24h 전 | 토스트 알림 |
| 마감 1h 전 | 팝업 + 경고음 |
| 마감 15분 전 | **풀스크린 차단** (닫기 불가) |
| 마감 초과 | 10분마다 반복 |

---

## 실행 방법

```bash
pip install -r requirements.txt
python main.py
```

Python 3.11 이상 권장. Windows 전용.
